# raman-open-forge-imai
A free, open-source desktop application for controlling low-cost Hamamatsu CCD Raman spectrometers, acquiring, processing spectra, creating and managing spectral databases (DBs) and performing basic spectra search in the DBs. The app is also adapted to work with ramanbase.org database materials. The code to assembly your own custom databases from ramanbase.org for working in the app is provided. Can be used with DIY Raman system based on Hamamatsu CCDs.
